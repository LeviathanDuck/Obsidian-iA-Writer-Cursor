# Fat Cursor

An iA Writer-style fat blue caret for Obsidian's editor. Replaces the default thin caret with a thicker, more visible one.

*A project of the Leviathan Duck from Leftcoast Media House Inc.*

## Inspired by iA Writer

This plugin exists of my love for [iA Writer](https://ia.net/writer) and its meticulous design choices — the distinctive fat caret is one of the small things that makes writing in iA Writer feel different. iA Writer and the team behind it deserves all the credit in the world.

If you haven't tried iA Writer, I'd encourage you to.

This plugin is an independent implementation and has no affiliation of any kind. No iA Writer code is used.

## Install

Either symlink this folder into a vault's `.obsidian/plugins/` directory, or copy `main.js`, `manifest.json`, and `styles.css` into `<vault>/.obsidian/plugins/fat-cursor/`. Then enable Community Plugins in Obsidian and turn on **Fat Cursor**.

## Development

This plugin is simple enough to not need a build pipeline — it's plain JavaScript and CSS. Edit `main.js` and `styles.css` directly, then reload the plugin in Obsidian.
